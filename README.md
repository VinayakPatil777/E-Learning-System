# Learning Management System (LMS)

## Project Overview
This is a full-stack E learning System developed using React.js and Spring Boot. The platform allows users to register, log in, and access learning-related features through a secure authentication system.

## Tech Stack
- Frontend: React
- Backend: Spring Boot, Spring Security
- Database: MySQL
- Authentication: JWT

## Features
- User Login and Registration
- JWT-based Authentication
- Secure REST APIs
- Frontend integration with backend
